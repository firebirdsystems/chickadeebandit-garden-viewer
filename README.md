# Map My Garden

A [Chickadee Bandit](http://chickadeebandit.com) app.

Browse your household gardens and plants on an interactive map. Add and manage plants from the [Map My Garden](https://mapmygarden.app) iOS app — this viewer keeps everyone in the family in sync.

## Features

- Interactive Leaflet map showing garden beds and plant positions
- Plant detail view with care notes and photos
- Read-only for family members — editing happens in the iOS app
- Syncs automatically when the Map My Garden iOS app updates

## Install

In your hub, go to **Apps → Install from URL** and paste:

```
https://github.com/firebirdsystems/chickadeebandit-garden-viewer/releases/latest/download/bundle.json
```
